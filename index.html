Wiki Global Contributions

Global Wiki Contributions tool is a search interface that lets users view a specific user's contributions across various wiki projects. It allows for filtering contributions by namespace, start date, and end date.

Wiki Anniversary

Find user anniversaries on Wikimedia projects and wish the user :)

Wikidata RC Map

Displays live changes made to Wikidata on a map. It shows the location of the change with coordinates, along with the sitelink and the QID. This allows users to visualize and track real-time edits to the database across the globe.

Panchayats of Kerala

The dashboard provides an overview of the local governing bodies in the state. It presents information sourced from Wikidata and includes various official identification codes for these bodies, such as the LSG, LGD and SEC Codes.

Wiki User Info Fetcher

Bulkuserinfo is a tool designed to check the contributions and user rights of multiple users at the same time. It allows users to efficiently retrieve a comprehensive summary of edit count and permissions for a list of accounts.

Kerala COVID-19 Data

A project that provides a district-wise visualization of the COVID-19 spread in Kerala. It contains daily updated data on active cases, confirmed cases, samples taken, negative samples, and hospitalizations, with some of the data added to Wikidata.

COVID-19 India Dashboard

This was an initiative dasashboard visualizes COVID-19-related data for India and its states and union territories. This is powered by data from Wikidata and was collaboratively contributed by volunteers
